# Social Network

Un social network che combina elementi di Instagram e BeReal.

## Funzionalità Principali

- Registrazione e Login degli Utenti
- Creazione e Condivisione di Post (con foto e video)
- Feed degli Amici
- Notifiche in Tempo Reale
- Messaggistica Privata
- Funzioni di BeReal (es. post temporizzati)

## Tecnologie Utilizzate

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
- **Database**: MongoDB
- **Autenticazione**: JWT (JSON Web Tokens)
